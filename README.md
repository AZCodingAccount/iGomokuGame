# iGomokuGame

## 介绍📘

​		iGomokuGame是一个基于Vue3、SpringBoot3，Element-Plus、WebSocket、SpringTask、DataV、五子棋AI算法等实现的一款在线五子棋游戏。支持**排行榜功能**、**好友功能**、**人机对战**、**联机对战**、**数据可视化**等功能。项目使用最新的技术开发，开发过程规范，逻辑严谨。
